# Hello, My Name is gogogo137! 🚀

Welcome to my personal dynamic greeting space. This project features a high-fidelity, interactive, and responsive web card showcasing a personalized greeting, a live clock widget, and a theme switcher.

---

## 🌟 Features

- **Dynamic Personalized Greeting:** Greets visitors with a personalized badge that dynamically updates depending on the time of day (*Good Morning*, *Good Afternoon*, or *Good Evening*).
- **Real-Time Live Clock:** Displays the current local time (HH:MM:SS AM/PM), local calendar date, and detects your local timezone automatically.
- **Futuristic Glassmorphic Design:** Styled with state-of-the-art CSS glassmorphism (`backdrop-filter`), vibrant animated background glow blobs, and elegant typography (using Google Fonts: *Outfit* & *Plus Jakarta Sans*).
- **Sleek Theme Switcher:** Seamlessly switch between a deep, rich dark space theme and a clean, vibrant light mode.
- **Fully Responsive Layout:** Optimized for all screen sizes, from mobile devices to desktop monitors.

---

## 🛠️ Technology Stack

- **Markup:** HTML5 (Semantic Structure)
- **Styling:** CSS3 (Custom Variables, Flexbox, Animations, Backdrop Filter)
- **Icons:** FontAwesome v6
- **Fonts:** Google Fonts API
- **Logic:** Vanilla JavaScript (ES6)

---

## 🚀 How to Run Locally

Since this is a client-side web page, you can run it directly:

1. Clone or download this repository.
2. Open [index.html](index.html) in any modern web browser (Chrome, Firefox, Safari, Edge).
